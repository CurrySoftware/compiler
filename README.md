# The Elm Compiler

Learn about the Elm programming language at [elm-lang.org](http://elm-lang.org/).


## Install

Follow [these instructions][installer]!

 [installer]: https://guide.elm-lang.org/install.html

## Build

With Stack:

1. Install Stack (https://www.haskellstack.org)
2. Clone the repo
3. `cd compiler`
4. `stack init`
5. `stack build`
